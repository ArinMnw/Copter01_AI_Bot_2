# ท่าที่ 13 — EzAlgo V5 (Supertrend)

| | |
|---|---|
| ไฟล์หลัก | `strategy13.py` (place orders: `_place_s13_split_orders` ใน `scanner.py`) |
| ประเภท | standalone |
| Default | OFF |
| Comment | market `{TF}_S13_EZ_#1..#3` · limit `{TF}_S13_EZ_L1..L3` |

## แนวคิด

- ใช้ **supertrend crossover** เป็นสัญญาณหลัก ใช้ได้ทุก TF ที่เปิด scan
- เมื่อเจอสัญญาณ → เลือก mix ของ market/limit ตามความสัมพันธ์ `current price` กับ `entry`
- ถ้าเจอสัญญาณฝั่งตรงข้าม → ล้าง exposure ของ S13 **เฉพาะ TF เดียวกัน** ก่อนเปิดฝั่งใหม่

## Config

`S13_SENSITIVITY`, `S13_SUPERTREND_ATR`, `S13_STOP_ATR_LEN`, `S13_STOP_ATR_MULT`,
`S13_TP1_RR`, `S13_TP2_RR`, `S13_TP3_RR`

## Signal

- BUY: `close` ตัดขึ้นเหนือ supertrend
- SELL: `close` ตัดลงใต้ supertrend

## Entry / SL / TP

- `entry` = ราคาปิดของแท่งสัญญาณ
- `SL`: BUY = `signal_low − ATR(stop)×mult` · SELL = `signal_high + ATR(stop)×mult`
- `TP`: `TP1 = 0.7R`, `TP2 = 1.2R`, `TP3 = 1.5R`

## Order flow

**BUY**
- `current price > entry` → market 1 order (`#3→TP3`) + limit 3 orders (`L1→TP1, L2→TP2, L3→TP3`)
- `current price <= entry` → market 3 orders (`#1→TP1, #2→TP2, #3→TP3`) + limit 1 order (`L3→TP3`)

**SELL**
- `current price < entry` → market 1 order (`#3→TP3`) + limit 3 orders (`L1/L2/L3`)
- `current price >= entry` → market 3 orders (`#1/#2/#3`) + limit 1 order (`L3→TP3`)

## Standalone behavior

- **ไม่เข้า** flow กลาง: Entry Candle, Trail SL, Opposite Order, Trend Filter, Limit Guard
- **ใช้** (เปิดตั้งแต่ 2026-05-18): Limit Trend Recheck (cancel pending ถ้าสวนเทรนด์), Fill RSI Recheck (ปิด position ถ้าไม่ผ่าน)

## Triple Scale-Out (TSO) — Always 4 Orders

- S13 ใช้ TSO สร้าง orders แยก **4 ชุดเสมอ** (ไม่ใช่ partial close)
- general formula: `[min(200pt,TP), min(300pt,TP), min(600pt,TP), TP]`
- **ไม่ขยาย lot** (แต่ละ order ใช้ `get_volume()` ปกติ) · **ไม่ลงทะเบียนใน `scale_out_state`** → toggle OFF ไม่ลบ S13 orders
- ทำใน `_place_s13_split_orders` · log `S13_TSO_TP_OVERRIDE`
- `SCALE_OUT_ENABLED = False` → ใช้ tp_levels RR-based ปกติ (3 orders)
- หมายเหตุ: จำนวน orders ของ S13 อาจไม่เป็น 3 เสมอ (1–4 ตาม TP เดิม)
