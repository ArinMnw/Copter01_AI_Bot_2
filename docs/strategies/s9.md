# ท่าที่ 9 — RSI Divergence

| | |
|---|---|
| ไฟล์หลัก | `strategy9.py` |
| ประเภท | standalone (bypass trend filter) |
| Default | ON |
| sync | `mql5/RSIDivergencePane.mq5` |

## แนวคิด

- ใช้ `RSI(close)` ชุดเดียวกับ pane `RSIDivergencePane.mq5` (default `RSI(14)`)
- ไม่ใช้ indicator divergence สำเร็จรูป — ใช้ **pivot RSI** แบบเดียวกับ TradingView / MQL5 pane
- เจอ divergence → ตั้ง **LIMIT @ midpoint** ของแท่ง pivot ปัจจุบันทันที

## แหล่งข้อมูล RSI (ต้อง sync กับ MQL5)

- period: `config.RSI9_PERIOD` · applied price: `config.RSI9_APPLIED_PRICE` (ล็อกเป็น `close`)
- ถ้าจะเปลี่ยน period ต้องแก้ทั้ง `config.py` **และ** compile `RSIDivergencePane.mq5` ใหม่ให้ตรงกัน

## Pivot detection

- pivot left = `RSI9_LEFT` (default 5), pivot right = `RSI9_RIGHT` (default 5)
- หา pivot คู่ติดกัน (**immediate previous pivot only**) ภายใน range `RSI9_RANGE_MIN..RSI9_RANGE_MAX`

## Divergence types (4 แบบ)

| toggle | default | ชนิด | เงื่อนไข | signal |
|---|---|---|---|---|
| `RSI9_PLOT_BULLISH` | ON | Regular Bullish | price LL + RSI HL | BUY |
| `RSI9_PLOT_BEARISH` | ON | Regular Bearish | price HH + RSI LH | SELL |
| `RSI9_PLOT_HIDDEN_BULLISH` | OFF | Hidden Bullish | price HL + RSI LL | BUY |
| `RSI9_PLOT_HIDDEN_BEARISH` | OFF | Hidden Bearish | price LH + RSI HH | SELL |

ปุ่ม Telegram รวมเหลือ 2 ปุ่ม: **Regular** (Bullish+Bearish) / **Hidden** (Bullish+Bearish)

## Entry / SL / TP

`order_mode = "limit"` (เปลี่ยนจากเวอร์ชันเก่าที่เป็น STOP) — entry = midpoint ของแท่ง pivot ปัจจุบัน

BUY (BUY LIMIT):
- `entry = (cur_high + cur_low) / 2`
- `SL = cur_low − SL_BUFFER`
- `TP` = swing TP / fallback RR 1:1

SELL (SELL LIMIT):
- `entry = (cur_high + cur_low) / 2`
- `SL = cur_high + SL_BUFFER`
- `TP` = swing TP / fallback RR 1:1

## Standalone behavior

- **bypass Trend Filter** (scan loop)
- **skip RSI Fill Recheck** (`sid in (1, 9, 11, 14, 15)`)
- **skip Fill Trend Recheck** (`sid in (9, 10, 14, 15)`)
- **skip PD Fibo Plus** (`sid in (9, 15)`)
- อยู่ใน `STRONG_TREND_BLOCK_SIDS` (เปิดกันไม้สวน strong trend ได้)

## หมายเหตุ

- `setup_sig` ใช้แค่ pivot identity (`tf|signal|div_type|pivot_prev_time|pivot_cur_time`) เพื่อกัน duplicate
- ถ้า pane RSI บน MT5 กับ bot ให้ค่าไม่ตรง → เช็ค period เป็นอย่างแรก
- code Python กับ MQL5 ทำงานแยกกันแต่ใช้สูตรเดียว → ผลตรงกันบน RSI ชุดเดียว
- เทียบ chart: log `bot.log` เป็น BKK, MT5 chart เป็น broker time
