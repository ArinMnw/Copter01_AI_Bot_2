# ท่าที่ 4 — นัยยะสำคัญ FVG

| | |
|---|---|
| ไฟล์หลัก | `strategy4.py` |
| ประเภท | main-flow |
| Default | ON |

## แนวคิด

- ต้องเป็น FVG ที่ **กลืน swing สำคัญจริง** ไม่ใช่แค่ gap ธรรมดา
- ต่างจากท่า 2 ตรงที่ต้องมี swing high/low ก่อนหน้าอยู่ "ใน gap" จริง และแท่งสัญญาณต้องปิดทะลุ swing นั้น
- เป็นที่มาของ swing helper ที่ท่า 8 นำไปใช้ต่อ

## Pattern (signal)

### BUY

- `[1]` เขียว และ `High[1] > High[2]`
- `[0]` ต้องยังไม่ปิด gap (เช่น `Low[0] > High[2]`)
- ต้องหา `Swing High` ก่อนหน้าให้เจอ
- `Close[1]` ต้องปิด **เหนือ** swing นั้น และห่างจาก swing ≥ `engulf_min_price()`
- swing ที่กลืนต้องอยู่ "ใน gap" จริง

### SELL — สลับเงื่อนไขตรงข้าม

- `[1]` แดง และ `Low[1] < Low[2]`
- `[0]` ต้องยังไม่ปิด gap
- `Close[1]` ต้องปิด **ต่ำกว่า** `Swing Low` ก่อนหน้า และห่างจาก swing ≥ `engulf_min_price()`

## หมายเหตุ

- `get_structure()` ของ S4 ใช้ ATR กลาง (`calc_atr`, True Range + RMA)
- swing high/low ที่ S4 หาได้ ถูกนำไปใช้เป็นฐานของ **ท่า 8 (กินไส้ Swing)** ดู [s8.md](s8.md)
