## แนวคิด

ท่า scalping ที่ใช้ 3 แท่ง `[2]`, `[1]`, `[0]` หา 2 รูปแบบ (Momentum / Reversal)
ทั้ง BUY/SELL แล้วกรองด้วย **4 ฟิลเตอร์** ก่อนเข้า (`_check_filters`)

> หมายเหตุ: source `strategy5.py` ปัจจุบัน implement ครบ (ต่างจาก stub เดิมใน `strategies.md`)

## 4 ฟิลเตอร์ (`_check_filters`)

1. **Time** — ห้ามเทรดในช่วง `S5_NO_TRADE_HOURS` (default `[(0, 3)]` = 00:00–03:00 BKK)
2. **ATR** — range แท่งล่าสุดต้องไม่เกิน `avg_atr × S5_ATR_MAX_MULT` (default 2.5) — กันเข้าแท่งผันผวนจัด
3. **Trend (EMA20)** — BUY ต้อง `Close[-2] ≥ EMA20`, SELL ต้อง `Close[-2] ≤ EMA20` (`S5_TREND_BARS = 20`)
4. **Zone (Swing H/L)** — entry ต้องห่าง Swing High (BUY) / Swing Low (SELL) อย่างน้อย `atr × S5_ZONE_BUFFER` (default 1.5); override ด้วย HHLL swing ถ้าระบุ `tf`

## Pattern

### BUY Momentum

- `[1]` เขียว body ≥ 60%
- `[0]` เขียว และ `Open[0] ≥ Close[1] − atr×0.1`
- `entry = Open[0]` · `SL = min(Low[0], Low[1]) − atr×0.5` · `TP = entry + atr×1.0`

### BUY Reversal

- `[2]` แดง body ≥ 35%
- `[1]` เป็น Doji (body < 15%) **หรือ** "ตำหนิ" (ไส้กินเข้า zone ของ `[2]`)
- `[0]` เขียว body ≥ 35% และ `Close[0] > High[1]`
- `entry = Low[1] + (High[1]−Low[1])×0.5` · `SL = Low[1] − atr×0.5` · `TP = entry + atr×1.5`

### SELL Momentum

- `[1]` แดง body ≥ 60%
- `[0]` แดง และ `Open[0] ≤ Close[1] + atr×0.1`
- `entry = Open[0]` · `SL = max(High[0], High[1]) + atr×0.5` · `TP = entry − atr×1.0`

### SELL Reversal

- `[2]` เขียว body ≥ 35%
- `[1]` Doji หรือ "ตำหนิ"
- `[0]` แดง body ≥ 35% และ `Close[0] < Low[1]`
- `entry = High[1] − (High[1]−Low[1])×0.5` · `SL = High[1] + atr×0.5` · `TP = entry − atr×1.5`

## Config (ค่าในไฟล์)

| key | default | ความหมาย |
|---|---|---|
| `S5_ATR_MAX_MULT` | 2.5 | เพดาน range แท่งล่าสุด (× avg ATR) |
| `S5_ZONE_BUFFER` | 1.5 | ระยะห่างขั้นต่ำจาก swing (× ATR) |
| `S5_TREND_BARS` | 20 | period EMA + จำนวนแท่ง trend |
| `S5_NO_TRADE_HOURS` | `[(0, 3)]` | ช่วงห้ามเทรด (ชม. BKK) |

## หมายเหตุ

- ต้องมีอย่างน้อย 25 แท่งถึงจะ scan (`len(rates) < 25` → WAIT)
- `reason` ของ WAIT ระบุชั้นฟิลเตอร์ที่ fail ชัดเจน (เช่น `📉 Trend: ...`, `🚧 Zone: ...`)
