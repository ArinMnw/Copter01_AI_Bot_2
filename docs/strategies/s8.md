# ท่าที่ 8 — กินไส้ Swing

## แนวคิด

- ใช้ swing high และ swing low ที่หาได้จาก `strategy4.py`
- ตั้ง order ได้ **ทั้งสองฝั่งในรอบเดียว** (ตั้ง limit ที่ขอบ swing ทั้งบนและล่าง)
- เป็นการดักไม้ที่ราคามา "กินไส้" ของ swing แล้วเด้งกลับ

## Entry / SL / TP

### SELL (ใช้ Swing High)

- `Entry = High + 17% ของ range swing`
- `SL = High + 31% ของ range swing`
- `TP = Swing Low`

### BUY (ใช้ Swing Low)

- `Entry = Low − 17% ของ range swing`
- `SL = Low − 31% ของ range swing`
- `TP = Swing High`

## Order flow (สำคัญ)

- S8 **ตั้ง limit ก่อน** แล้ว SL อาจถูก arm ทีหลังตาม flow ปัจจุบัน
- ถ้า S8 **fill ก่อน arm SL** → ต้องมี fallback ไปตั้ง SL ให้ position ที่ fill แล้ว
- S8 ข้าม adjacent-bar check (ตั้งแท่งติดกันได้) — `sid != 8` ใน scanner

## ความสัมพันธ์กับ Limit Sweep

- ถ้า S8 มาจาก **Limit Sweep** ต้องระวังบริบทของ `LL/HH`
- ห้ามใช้ swing ที่แท่งปิดทะลุผ่านไปแล้ว — ต้องเลือก level ที่ยัง valid

## หมายเหตุ

- swing high/low มาจากตัวเดียวกับท่า 4 ([s4.md](s4.md))
