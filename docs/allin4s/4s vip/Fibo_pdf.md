## ส่วนที่ 2: การวิเคราะห์พฤติกรรมราคา (Price Action) และกราฟแท่งเทียน

เอกสารนี้อธิบายถึง **ระบบการเทรดด้วยแท่งเทียนควบคู่กับ Fibonacci ระดับพิเศษ (Custom Fibonacci Levels)** โดยมีรายละเอียดเชิงลึกของพฤติกรรมราคาดังนี้:

### 1. รูปแบบแท่งเทียนที่เป็นจุดเริ่มต้นสัญญาณ (Signal Candlestick Patterns)
ระบบนี้ให้ความสำคัญกับการหาจุดเข้าจากแท่งเทียน 2 รูปแบบหลัก ซึ่งจะใช้เป็น "จุดอ้างอิง (Anchor)" ในการกาง Fibonacci (ลากจากจุด 0 ถึง 1 หรือฐานของแท่งสัญญาณ)
- **แท่งกลืนกิน (Engulfing):** เกิดขึ้นเมื่อมีแท่งเทียนสีตรงข้าม (เช่น สีเขียว) ปิดคลุมได้ทั้ง "เนื้อเทียน" และ "ไส้เทียน" ของแท่งก่อนหน้า (สีแดง) แบบมิดชิด บ่งบอกถึงแรงซื้อหรือขายที่ชนะขาดลอย
- **แท่งตำหนิ (Defect):** เกิดขึ้นเมื่อแท่งเทียนสีตรงข้ามสามารถปิดคลุมได้เพียงแค่ "เนื้อเทียน" แต่ไม่สามารถคุม "ไส้เทียน" ได้ทั้งหมด (ทิ้งไส้โผล่ออกมา) 
- *ข้อแม้สำคัญ:* การจะคอนเฟิร์มว่าราคาจะไปต่อในทิศทางนั้นจริงๆ (เช่น ขึ้นจริง) จะต้องมีแท่งเทียนสีเดียวกัน (เช่น สีเขียว) ตามมาอีก 1 แท่ง รวมเป็น 2 แท่งยืนยัน และหากแท่งกลืนกินมีขนาดใหญ่เกินไปก็ถือว่าไม่ดี ควรหาแท่งที่มีขนาดพอเหมาะ

### 2. ระดับของ Indicator Fibonacci และความหมาย (Key Support/Resistance & Targets)
มีการตั้งค่าระดับ Fibo แบบกำหนดเอง ซึ่งแต่ละระดับจะทำหน้าที่เป็นแนวรับ-แนวต้าน (Support & Resistance) หรือเป้าหมายในการทำกำไร:
- **โซน 0.4 ถึง 0.57 (โซน 50-60%):** เป็นโซนสำหรับพักตัวหรือทดสอบแนวรับหลังจากการเบรกเอาท์
- **1.617 (เรียกว่า KRH1 หรือ "ระดับ 1"):** ทำหน้าที่เป็น FVG (Fair Value Gap)
- **3.097 (เรียกว่า KRH2 หรือ "ระดับ 2"):** ทำหน้าที่เป็นโซนแนวรับ-แนวต้านที่แข็งแกร่ง (Support and Resistance)
- **5.165 (เรียกว่า KRH3 หรือ "ระดับ 3"):** ทำหน้าที่เป็นโซน FVG, DM (Demand), SP (Supply) ซึ่งต้องมีโครงสร้างอื่นมาคอนเฟิร์มร่วมด้วยในโซนเดียวกัน
- **7.044 (เรียกว่า RUN):** เป็นเป้าหมายหลักของการวิ่งรอบใหญ่ (Major Target)
- **8.237 (เรียกว่า X):** เป็นระดับที่มักจะเกิดสัญญาณ Indicator RSI Divergence (สัญญาณขัดแย้งของ RSI ที่บอกถึงการกลับตัว)

### 3. พฤติกรรมราคาและแนวโน้มที่ปรากฏในระบบ (Trend Scenarios & Price Action)
เอกสารอธิบาย Flow การเคลื่อนที่ของราคาตามระดับ Fibo เป็นฉาก (Scenarios) ต่างๆ ดังนี้:

**พฤติกรรมในขาขึ้น (Bullish Scenarios)**
1. **ชน 1 เทส 50:** ราคาเบรกขึ้นไปชนระดับที่ 1 (1.617) แล้วย่อลงมาทดสอบโซน 50% (ราวๆ 0.4 - 0.57) หากรับอยู่ (ไม่หลุด) ราคาจะมีโอกาสวิ่งไปถึงเป้าหมาย RUN (7.044) แต่ถ้าหลุด 50% ราคาจะกลับลงไปหา Low เดิม
2. **ชน 2 เทส 50-60:** วิธีการที่ง่ายที่สุดของระบบ คือหาแท่งกลืนกิน/ตำหนิให้เจอ ราคาจะขึ้นไปชนระดับ 2 (3.097) แล้วย่อลงมาลึกถึงโซน 50-60% เพื่อสร้างฐาน หากรับอยู่จะวิ่งยาวไปถึง RUN
3. **ชน RUN แล้วย่อกลับมาเทส 2:** เมื่อราคาวิ่งถึงเป้า RUN (7.044) มักจะมีการพักตัวลึกกลับลงมาหาแนวรับระดับ 2 (3.097) หากราคาสามารถ "ยืนเหนือระดับ 2 ได้" (สร้างฐานและไม่หลุด) ราคาจะกลับตัวขึ้นไปทะลุเป้า RUN เดิมเพื่อไปสู่ระดับที่สูงกว่า
4. **ชน 3 เทส 1 ยืน 2 ได้:** ราคาวิ่งขึ้นไปถึงระดับ 3 (5.165) แล้วย่อแรงลงมาทดสอบระดับ 1 (1.617) หากสามารถดีดกลับขึ้นไปและ "ยืนเหนือระดับ 2 (3.097)" ได้ จะกลับไปวิ่งหา RUN

**พฤติกรรมความล้มเหลวและขาลง (Bearish Failure / Breakdown Scenarios)**
1. **ยืน 2 ไม่ได้ (คุยกับรากมะม่วง):** หากราคาวิ่งไปถึงเป้าหมายสูงๆ (เช่น RUN) แล้วย่อลงมาที่ระดับ 2 (3.097) แต่ไม่สามารถยืนเป็นแนวรับได้ (ราคาทะลุหลุดลงไป) จะเกิดการเทขายอย่างหนัก ทะลุจุดเริ่มต้น 0 ลงไปทำ New Low
2. **3 เทส 1 ยืน 2 ไม่ได้:** ราคาไปถึงระดับ 3 ย่อมาที่ 1 แล้วเด้งไปที่ 2 แต่ไม่ผ่านระดับ 2 (ถูกปฏิเสธ) ราคาจะร่วงหนักทะลุฐาน
3. **RUN เทส 1 ยืน 2 ไม่ได้:** ในทำนองเดียวกัน หากไปถึง RUN แล้วร่วงมาที่ 1 เด้งขึ้นไปชนระดับ 2 แต่ไม่ผ่าน จะเกิดการเทขายอย่างรุนแรง (เอกสารเตือนว่า "เจอเทหนักอย่าหาซ้อน (สวน)")

**กฎเฉพาะตัว:**
- การจะใช้ Fibo ระบบนี้ให้แม่นยำ กราฟจะต้องมีการสร้าง "High-Low" ที่ชัดเจนเป็นของตัวเอง
- ถ้ากราฟพุ่งขึ้นไปเป็นเส้นตรงยาวรวดเดียวจนถึงระดับ 3 (5.165) โดยไม่มีการพักตัวเลย ถือว่าเป็นรูปแบบที่ไม่สามารถเล่นตามระบบนี้ได้
- โครงสร้างนี้ใช้ได้กับทั้งขาขึ้นและขาลง (ในหน้า 16 มีตัวอย่างของคู่ทองคำ XAU ที่ตี Fibo กลับหัวสำหรับขาลง โดยมีเป้า RUN อยู่ด้านล่าง)

## ส่วนที่ 3: แผนภาพและการทำงานของท่าย่อย S20.5.Fibo_Entry (อัปเดตล่าสุด)

ท่าย่อย **S20.5.Fibo_Entry** ถูกพัฒนาและอัปเดตจาก PDF โดยแบ่งจุดเข้าหลักๆ ออกเป็น 4 รูปแบบ (Sub-Patterns) ดังนี้:

### 1. S20.5.Fibo_Entry_Hit1 & Hit2 (Test 50% Zone)
- **จังหวะเริ่มต้น (Setup):** ราคาเบรกระดับ 1.617 (Hit 1) หรือระดับ 3.097 (Hit 2) ขึ้นไป
- **จังหวะย่อตัว (Pullback & Reject):** ราคาย่อลงมาที่โซน 50%-60% (Fibo 0.4 ถึง 0.57) เมื่อแท่งเทียนปิดตัว จะต้องเห็นลักษณะ "การปฏิเสธราคา" (ทิ้งไส้ / Reject) 
- **จุดเข้า (Entry):** ที่ระดับ 50% ของรอบ
- **เป้าหมาย (Target):** ระดับ RUN (7.044)

### 2. S20.5.Fibo_Entry_3Test1 (ชน 3 เทส 1)
- **จังหวะเริ่มต้น (Setup):** ราคาวิ่งทะลุขึ้นไปจนชนแนวระดับ 3 (5.165) 
- **จังหวะย่อตัว (Pullback & Reject):** ราคาย้อนกลับลงมาแรงๆ เพื่อทดสอบฐานระดับ 1 (1.617) และมีแท่งเทียนทิ้งไส้ปฏิเสธโซนรับนี้ (ยืน 2 ได้ในอนาคต แต่เข้าจังหวะเด้ง)
- **จุดเข้า (Entry):** ที่ระดับ 1 (1.617)
- **เป้าหมาย (Target):** ระดับ RUN (7.044)

### 3. S20.5.Fibo_Entry_RunTest2 (ชน RUN เทส 2)
- **จังหวะเริ่มต้น (Setup):** ราคาวิ่งขึ้นไปถึงเป้าหมายหลัก RUN (7.044)
- **จังหวะย่อตัว (Pullback & Reject):** ราคาถูกเทขายอย่างหนักกลับลงมาที่ระดับ 2 (3.097) และมีการสร้างแท่งเทียนทิ้งไส้ (ยืน 2 ได้) เพื่อสะสมพลังกลับขึ้นไป
- **จุดเข้า (Entry):** ที่ระดับ 2 (3.097)
- **เป้าหมาย (Target):** ทะลุกลับขึ้นไปหาเป้า RUN อีกครั้ง หรืออาจจะสูงกว่าเดิม

### ข้อควรระวังและการตัดขาดทุน (Warnings, SL & Invalidations)
- **ขึ้นเป็นเส้นตรงทะลุ 3 (Straight Line to 3 Invalidation):** ตามเอกสารระบุชัดเจนว่า "ถ้าขึ้นเป็นเส้นตรงเลย 3 จะเล่นไม่ได้" หมายความว่าถ้าราคาวิ่งจากจุดฐานพุ่งทะลุระดับ 3 (5.165) โดยไม่มีการย่อพักตัวใดๆ (ไม่มีแท่งสีตรงข้ามสลับเลย) จะถือเป็นโครงสร้างที่ใช้งานไม่ได้ (Invalid Setup) ห้ามเข้าเทรด
- **คุยกับรากมะม่วง (Mango Root Fall):** หากเกิดรูปแบบ "ชน RUN ย่อลงมาหา 2 แล้วยืน 2 ไม่ได้" (ไม่มีการปฏิเสธราคา ทะลุ 3.097 ลงไป) ห้ามเข้าเทรดเด็ดขาด เพราะราคาจะเทขายอย่างหนักทะลุฐาน 0 หรือถ้าราคาพุ่งชน RUN แล้วร่วงมาทดสอบระดับ 1 (RUN เทส 1) เอกสารเตือนว่า "เจอเทหนักอย่าหาซ้อน" (ห้ามช้อนซื้อ)
- **3 เทส 1 ยืน 2 ไม่ได้:** หากราคาย่อจาก 3 มา 1 แล้วเด้งขึ้นไป 2 แต่ยืนเหนือ 2 ไม่ได้ (ปิดต่ำกว่า) ราคาจะร่วงทะลุฐาน 0 เช่นกัน
- **สถิติความน่าจะเป็น (Win Rates & Probabilities):** 
  - **ชน 1 เทส 50:** โอกาสทะลุผ่าน (Breakout Probability) อยู่ที่เพียง 50% (50-50) ซึ่งถือว่ามีความเสี่ยงระดับปานกลาง 
  - **ชน 2 เทส 60-50-Krl:** หากราคาวิ่งชน 2 แล้วย่อตัวลงมาทดสอบโซน 50-60% ลึกไปจนถึง KRL (ระดับ 0) แต่ไม่หลุด และมีการทิ้งไส้ปฏิเสธ จะมีโอกาสกลับตัวทะลุขึ้นไปทำกำไรสูงถึง 90% ซึ่งเป็น Setup ที่มี Win Rate สูงที่สุดในระบบนี้
- **SL Position:** วาง Stop Loss เผื่อหลอก (Buffer) ไว้ต่ำกว่าจุด 0 (KRL) ของฐานชุด Fibo เสมอ เพื่อป้องกันการสะบัดกิน Stop Loss
