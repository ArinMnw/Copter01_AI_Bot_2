**หน้า 2:**
**หัวข้อ: สำหรับใครที่งงแท่งตำหนิกับกลืนกินคืออะไร**
- **แท่งกลืนกิน (Engulfing):** คือแท่งเทียนสีเขียวที่เนื้อเทียนปิดคลุมไส้เทียนของแท่งแดงก่อนหน้าได้มิดชิด การจะยืนยันการขึ้นจริงต้องมีแท่งสีเขียว 2 แท่ง
- **แท่งตำหนิ:** คือแท่งเทียนสีเขียวที่ปิดไม่คลุมไส้ของแท่งแดง หรือทิ้งไส้ทั้งเนื้อนับว่าเป็นตำหนิ การจะยืนยันการขึ้นจริงต้องมีแท่งสีเขียว 2 แท่งเช่นกัน

**[วิเคราะห์ Price Action เชิงลึก]:**
- แท่งกลืนกินสมบูรณ์ (Full Engulfing) แสดงถึงโมเมนตัมของฝั่งซื้อ (Demand) ที่มีอำนาจเหนือแรงขาย (Supply) แบบเบ็ดเสร็จ กวาดเอา Liquidity เหนือไส้เทียนของฝั่งขายไปเรียบร้อยแล้ว
- ส่วนแท่งตำหนิ (Defect Candle) แสดงถึงความลังเล หรือการที่ฝั่งซื้อพยายามดันราคาขึ้นไปแต่ถูกสถาบัน (Institutional Sellers) ขวางไว้ ทำให้เกิดไส้เทียนหรือปิดไม่เต็มแท่ง อย่างไรก็ตาม ทั้งสองรูปแบบหากเกิดแท่งเขียวยืนยันแท่งที่สอง (Confirmation) จะสะท้อนว่าฝั่งซื้อเริ่มสะสมแรงได้สำเร็จและพร้อมจะเปลี่ยนโครงสร้างราคา (Change of Character)

**หน้า 3:**
**หัวข้อ: กลืนกิน และ ตำหนิ**
- แสดงภาพเปรียบเทียบชัดเจนระหว่างแท่งเทียนรูปแบบกลืนกิน (เขียวคลุมแดงมิด) และตำหนิ (เขียวปิดไม่คลุมหรือมีไส้ยาว)

**[วิเคราะห์ Price Action เชิงลึก]:**
- **แท่งกลืนกิน:** กราฟเบรกสวิงเดิมได้อย่างมีประสิทธิภาพ หมายความว่า Smart Money จงใจกวาด Stop Loss ขา Sell และพร้อมลากราคาขึ้นต่อ
- **แท่งตำหนิ:** การที่มีไส้แทงขึ้นไปแต่ปิดต่ำกว่า หมายถึงการพยายามไปหยิบ Liquidity ด้านบนแต่เจอแรงขายตบกลับ (Wick Rejection) การเทรดในรูปแบบตำหนิจึงต้องระมัดระวังกว่าและรอแท่งยืนยันเพื่อยืนยันว่าฝั่งซื้อมีแรงจริง ไม่ใช่แค่การล่อเม่าไปดอย (Bull Trap)

**หน้า 4:**
**หัวข้อ: (ฟีโบนี้ จะเก่งก็ต่อเมื่อ มี HIGH-LOW เป็นของตัวเอง) การคอนเฟิร์มจุดเข้าของ FIBO**
- การตั้งค่า Fibonacci และความหมายของแต่ละเลเวล:
  - **1.617 KRH1 FVG**
  - **3.097 KRH2 Support and resistance** (แนวรับแนวต้าน)
  - **5.165 KRH3 FVG DM SP** (ต้องมีอย่างอย่างอยู่ในโซนเดียวกัน)
  - **7.044 RUN FVG**
  - **8.237 X Indicator RSI Divergence**
- หมายเหตุ: สามารถตีได้ทั้งแท่งกลืนกินและตำหนิ

**[วิเคราะห์ Price Action เชิงลึก]:**
- การตีฟีโบนาชีกำหนดเลเวลต่างๆ ไม่ใช่แค่ตัวเลขวิเศษ แต่เป็นการหา **Liquidity Pools** และ **Fair Value Gaps (FVG)** ที่สถาบันทิ้งร่องรอยไว้
- เลเวลอย่าง 1.617 (KRH1) มักจะเป็นจุดที่มี Inbalance หรือ FVG ซึ่งราคาชอบย้อนกลับมาเติม (Mitigate) เพื่อรับของ
- เลเวล 3.097 (KRH2) เป็นจุดสะสมแรงที่รายย่อยมองว่าเป็นแนวรับแนวต้านทั่วไป (Retail S/R) มักจะเกิดการต่อสู้ของราคาที่โซนนี้
- โซนที่ได้เปรียบในการรอ Pending Order คือโซนที่มี FVG หรือ Demand/Supply (DM SP) ซ้อนทับกับเลเวล Fibo (เช่น 5.165 KRH3) เรียกว่ามี Confluence ซึ่งเพิ่ม Win Rate สูงขึ้นมาก

**หน้า 5:**
**หัวข้อ: ตัวอย่างการตี Fibo**
- กราฟซ้าย: การตี Fibo จากแท่งตำหนิ
- กราฟขวา: การตี Fibo จากแท่งกลืนกิน (มีโน้ตว่า "ถ้าแท่งใหญ่เกินไปก็ไม่ดี ต้องแท่งพอเหมาะ")

**[วิเคราะห์ Price Action เชิงลึก]:**
- การตี Fibo จากแท่งเทียนที่พอเหมาะ (ไม่ใหญ่เกินไป) เป็นการหาจุดเข้าที่มี Risk:Reward คุ้มค่าที่สุด หากแท่งเทียนจุดกลับตัวใหญ่เกินไปแปลว่าโมเมนตัมถูกปล่อยออกมามากแล้ว (Exhaustion) ทำให้จุดเข้าอยู่ไกลและ Stop Loss กว้าง 
- การใช้แท่งตำหนิเป็นจุดอ้างอิง Fibo แม้จะดูไม่สวยงาม แต่ในมุมมองสถาบัน มันคือจุดที่มีการสร้างและกวาด Liquidity ภายในแท่งเดียว ทำให้เลเวล Fibo ที่กางออกมามีนัยยะในการเป็นโซนพักตัว (Pullback zones)

**หน้า 6:**
**หัวข้อ: โอกาสทะลุเลเวล 1 ไป RUN**
- กฎ: 1 เทส 50 มีโอกาสหลุด 50-50 ถ้าไม่หลุดวิ่งถึง RUN
- ถ้า 1 เทส 50 แล้วหลุด กลับไปหา Low เก่า

**[วิเคราะห์ Price Action เชิงลึก]:**
- **การชนด่าน 1 (1.617):** เมื่อกราฟวิ่งขึ้นไปชนเลเวล 1.617 แล้วย่อตัวลงมาเทสที่โซน 50% ของรอบการขึ้น (หรือ 0.5-0.618) นี่คือกระบวนการ **Mitigation** สถาบันกำลังย้อนกลับมาเก็บของ (Reload) ในบริเวณ Discount Zone (ต่ำกว่า 50%)
- **ถ้าไม่หลุด 50%:** แสดงว่าสถาบันปกป้องฐานราคานี้ไว้เพื่อลากราคาขึ้นไปสู่เป้าหมายหลักคือเลเวล RUN (7.044) ซึ่งเป็นโซน Take Profit (Buy Side Liquidity)
- **ถ้าหลุด 50%:** หมายถึงแรงซื้อในรอบนั้นเป็นเพียงภาพลวงตา (Fake Structure) ราคาจะไหลลงไปหา Internal Liquidity หรือ Low เดิมเพื่อเคลียร์ Stop Loss ของรายย่อยที่ตั้งไว้ใต้โซน 50%

**หน้า 7:**
**หัวข้อ: หาแท่งตำหนิ/กลืนกิน แล้วเทส 50-60 วิ่งถึง RUN**
- "อันนี้ง่ายๆเลย หาแท่งตำหนิกับกลืนกินให้เจอ แล้วตี Fibo 2เทส 50-60 วิ่งถึง RUN"
- กราฟแสดงราคาขึ้นไปแตะเลเวล 2 (3.097) แล้วย่อลงมาที่โซน 50-60% ก่อนจะพุ่งขึ้นไปถึง RUN (7.044)

**[วิเคราะห์ Price Action เชิงลึก]:**
- การที่ราคาสามารถทะลุเลเวล 1 ไปถึงเลเวล 2 (3.097) ได้ แปลว่าโครงสร้างเทรนด์แข็งแรงมาก (Strong Momentum)
- การย่อกลับมาลึกถึงโซน 50-60% (Discount/Premium zone) คือการทำ **Inducement (การหลอกล่อ)** ทำให้รายย่อยคิดว่ากราฟเสียทรงและเริ่มแห่กันเซล (Sell)
- แต่ในความจริง โซน 50-60% คือโซนที่สถาบันรอเก็บของราคาถูก (Optimal Trade Entry - OTE) เมื่อกวาดสภาพคล่องรายย่อยฝั่ง Sell เรียบร้อย ราคาก็จะถูกกระชากขึ้นไปทำกำไรที่เป้า RUN ทันที ถือเป็น Setup ที่คลาสสิกและได้เปรียบมากที่สุดในการรอ Buy Limit

**หน้า 8:**
**หัวข้อ: ยืน 2 ได้วิ่งกลับ RUN**
- "จากภาพ 2เทส 50 และ ไป RUN ย้อนลงมาหา 2 ยืน 2 ได้ กลับไปทะลุ RUN"
- กราฟแสดงการขึ้นแตะ RUN (7.044) แล้วย่อแรงกลับลงมาที่เลเวล 2 (3.097) เมื่อยืนได้ ก็จะพุ่งกลับไปทำ High ใหม่ทะลุ RUN ไปสู่เป้าหมายถัดไป

**[วิเคราะห์ Price Action เชิงลึก]:**
- เมื่อราคาวิ่งถึงเป้าหมาย RUN (7.044) จะเกิดแรงเทขายทำกำไรอย่างหนักจากสถาบัน (Take Profit triggering) ทำให้ราคาย่อตัวอย่างรุนแรง (Deep Pullback)
- การย่อลงมาเทสเลเวล 2 (3.097) ซึ่งเคยเป็นแนวต้าน (Resistance) และเปลี่ยนหน้าที่กลายเป็นแนวรับ (Support) หรือที่เรียกว่า **S/R Flip หรือ Breaker Block**
- ถ้าราคา "ยืนได้" (สะสมกำลังและไม่หลุด) แปลว่าสถาบันใช้โซนนี้ในการ Re-accumulation (สะสมของรอบใหม่) เป้าหมายต่อไปคือการไปกวาด Liquidity เหนือยอด RUN เดิม (External Liquidity) ใครที่เปิดออเดอร์สวนเทรนด์ (Counter-trend) ตอนราคาย่อ จะถูกล้างพอร์ตตอนราคาทะลุ RUN ขึ้นไป

**หน้า 9:**
**หัวข้อ: ถ้ายืน 2 ไม่ได้**
- "ถ้ายืน 2 ไม่ได้กลับไปคุยกับรากมะม่วง"
- กราฟแสดงราคาที่ย้อนลงมาหาเลเวล 2 (3.097) แต่ไม่สามารถยืนได้ ทำให้ราคาร่วงลงไปยาว

**[วิเคราะห์ Price Action เชิงลึก]:**
- เลเวล 2 เป็นโครงสร้างสำคัญ (Key Structure) หากราคาทะลุกลับลงมาและยืนไม่ได้ (Break of Structure ลงมา หรือเกิด Market Structure Shift ขาลงชัดเจน) 
- เหตุการณ์นี้บ่งชี้ว่า **Demand หมดสภาพ** โซนที่ควรจะมีสถาบันมารับซื้อกลับไม่มีแรงซื้อหลงเหลืออยู่ การไหลทะลุลงมาคือการที่ Smart Money เปลี่ยนฝั่งมาเป็นขาย (Distribution) ทำให้ราคาดิ่งลงไปหารากมะม่วง (กวาด Liquidity ของฝั่ง Buy ที่อยู่ต่ำกว่าทั้งหมด)

**หน้า 10:**
**หัวข้อ: จากภาพ 3 ยืน 1 และยืน 2 ได้ ไป RUN**
- "ถ้า 3 ยังไม่แตะ RUN แปลว่ายังไม่จบรอบ"
- กราฟแสดงราคาวิ่งขึ้นไปชน 3 (5.165) แล้วย่อลงมาเทสที่ 1 (1.617) และยืนที่ 2 (3.097) ได้ สุดท้ายก็ไปถึง RUN

**[วิเคราะห์ Price Action เชิงลึก]:**
- เลเวล 3 (5.165) ถือเป็นโซนแนวต้านหลักที่แข็งแกร่ง (Major Supply Zone) เมื่อราคาชนย่อมเกิดการปรับฐานลึก (Complex Pullback)
- การที่ราคาย่อลึกไปถึงเลเวล 1 (1.617) เป็นการล้วงลึกไปถึงโครงสร้างฐานแรกสุดเพื่อค้นหา Liquidity ก้อนใหญ่ (Deep Mitigation) รายย่อยมักจะโดนตัดขาดทุน (Stop out) ไปแล้วในช่วงย่อลึกนี้
- การที่ราคากลับมายืนเหนือ 2 (3.097) ได้อีกครั้ง (Change of Character ย่อย) เป็นสัญญาณคอนเฟิร์มว่าการย่อลึกนั้นเป็นเพียงการกวาด Stop Loss (Liquidity Grab) และตราบใดที่ราคายังไม่ชน RUN สถาบันก็จะยังคงมี Order ที่ต้องการทำกำไรค้างอยู่ด้านบน (Unmitigated Target)

**หน้า 11:**
**หัวข้อ: 3 เทส 1 ยืน 2 ไม่ได้ทะลุ**
- กราฟแสดงราคาวิ่งไป 3 ย่อมา 1 พยายามเด้งขึ้นไปยืน 2 แต่ไม่ผ่าน แล้วร่วงลงต่อ

**[วิเคราะห์ Price Action เชิงลึก]:**
- เมื่อราคาย่อลึกจาก 3 มาเทส 1 แล้วพยายามเด้งกลับ การที่ชนเลเวล 2 (3.097) แล้วไม่ผ่าน (Failure Swing / Lower High) 
- แสดงให้เห็นถึง **การติด Supply Zone ใหม่ที่ถูกสร้างขึ้นตอนราคาลง** สถาบันไม่ได้ต้องการดันราคาไปต่อแล้ว แต่ใช้จังหวะเด้งนี้เพื่อ **ระบายของรอบสุดท้าย (Mark-down phase)** เมื่อเม่าเห็นกราฟเด้งแล้วรีบเข้า Buy ตาม ก็จะติดกับดักตรงเลเวล 2 และถูกทุบลงไปทำ New Low

---

**หน้า 12:**
**หัวข้อ: RUN เทส 1 ยืน 2 ได้กลับไป RUN**
- กราฟแสดงราคาวิ่งไปแตะ RUN ทิ้งตัวลงมาหาเลเวล 1 สามารถสะสมพลังกลับไปยืน 2 ได้ แล้วพุ่งกลับไปทำไฮใหม่ทะลุ RUN

**[วิเคราะห์ Price Action เชิงลึก]:**
- การชน RUN แล้วเทแรงจนถึงเลเวล 1 คือพฤติกรรมสุดโต่งของการทำกำไร (Exhaustion / Panic Sell) รายย่อยที่ Long มาจากด้านล่างจะถูกล้างกำไรจนหมดหรือโดนกิน Stop loss กวาดเรียบ
- แต่ถ้าราคาสามารถฟอร์มตัวกลับขึ้นมายืนเลเวล 2 (3.097) ได้ หมายความว่าสถาบันเห็นมูลค่า (Value) และทำการสะสม Position ใหม่อีกครั้ง (Re-accumulation) โซนเลเวล 2 จะทำหน้าที่เป็น Spring board ส่งราคาให้ทะลุแนวต้านเดิม กวาดยอดไฮเก่าไปทำสถิติใหม่ (Stop Run บน External Liquidity)

---

**หน้า 13:**
**หัวข้อ: RUN เทส 1 ยืน 2 ไม่ได้ จะลงต่อ**
- "เจอเทหนักอย่าหาซ้อนนะครับ"
- กราฟแสดงการชน RUN เทส 1 พยายามเด้งไป 2 แต่ไม่ผ่าน สุดท้ายเทกระจาดลงต่อ

**[วิเคราะห์ Price Action เชิงลึก]:**
- คล้ายหน้า 11 แต่รุนแรงกว่าเพราะจบรอบที่ RUN ไปแล้ว การเด้งจาก 1 ไป 2 เป็นเพียง **Dead Cat Bounce (การเด้งเพื่อลงต่อ)** 
- สถาบันกระจายของเสร็จสมบูรณ์แล้ว ไม่มีเหตุผลที่จะต้องปกป้องราคาอีก โซนเลเวล 2 กลายเป็นกำแพงเหล็ก (Institutional Supply) เมื่อรายย่อยพยายามถัว (ซ้อน) ออเดอร์ฝั่ง Buy ตอนกราฟเด้ง จะถูกกระแทกอย่างหนักจนล้างพอร์ต นี่คือพฤติกรรมโครงสร้างขาลงเต็มรูปแบบ

- ภาพจำลอง 3 รูปแบบ:
  1. ชน 1 (KRH1) มีโอกาสทะลุลงมาหาโซน 50%
  2. ชน 2 (KRH2) เทสโซน 50-60% ของ KRL มีโอกาสถึง 90%
  3. ชน 3 (KRH3) มักจะเทสลึกถึงเลเวล 1 (KRH1)

**[วิเคราะห์ Price Action เชิงลึก]:**
- ยิ่งราคาพุ่งชนด่านที่สูงขึ้น (จาก 1 ไป 2 ไป 3) **ปริมาณ Liquidity ฝั่งตรงข้ามที่สะสมไว้เพื่อเทคกำไร (Take Profit Orders) ก็จะยิ่งหนาแน่นขึ้น** 
- การที่ชน 3 แล้วต้องย่อลึกถึง 1 สะท้อนถึงกฎของแรงโน้มถ่วงในกราฟเทรด (Imbalance mitigation) ราคาต้องการกลับมาเติมเชื้อเพลิงให้เต็มถังที่ฐานเดิมก่อนจะออกเดินทางรอบใหม่

**หน้า 15:**
**หัวข้อ: กราฟขึ้นเป็นเส้นตรง และ กฎการแตะ RUN**
- "ถ้าขึ้นเป็นเส้นตรงเลย 3 จะเล่นไม่ได้"
- "RUN จะเทส 1. แต่ถ้าแตะ RUN แล้วผ่าน 2 เมื่อไหร่จะทะลุจุดที่ตีทันที"

**[วิเคราะห์ Price Action เชิงลึก]:**
- **กราฟขึ้นเป็นเส้นตรง:** คือพฤติกรรมไล่ราคา (Price Chasing) ที่ไม่มีการสร้างฐานหรือ Pullback ที่แข็งแรง โครงสร้างแบบนี้เปราะบางมาก (Low Resistance Liquidity Run) เมื่อสถาบันหยุดซื้อ ราคาจะพังทลายลงมาง่ายดาย การตี Fibo และหาจุดเข้าจากทรงนี้จึงอันตรายมาก
- **กฎแตะ RUN ผ่าน 2:** เมื่อราคาจบรอบใหญ่ (RUN) และเริ่มเปลี่ยนโครงสร้าง ถ้าด่านเลเวล 2 (Key Support) แตกเมื่อไหร่ มันคือการยืนยันว่า **เทรนด์ได้เปลี่ยนทิศทางแล้วอย่างสมบูรณ์ (Market Structure Reversal)** กราฟจะรูดทะลุลงไปหาจุดเริ่มต้นของการดึง Fibo อย่างรวดเร็ว เพราะไม่มีแนวรับที่มีนัยยะสำคัญหลงเหลืออยู่อีกต่อไป

**หัวข้อ: กฎการทะลุจุดที่ตี (ตัวอย่างกราฟจริง XAU)**

**[วิเคราะห์ Price Action เชิงลึก]:**
- นี่คือตัวอย่างของ **Failure Swing กลับทิศ** เมื่อราคาลงไปถึงเป้าหมาย RUN ของฝั่ง Sell (Liquidity ฝั่งขายถูกกวาดเกลี้ยงแล้ว) 
- เมื่อ Smart Money กลับตัวมาซื้อ การพังทลายของแนวต้านที่เลเวล 2 (KRH2 ฝั่งลง) ถือเป็นการทำลายโครงสร้างฝั่ง Sell ทิ้ง (Break of Structure) ออเดอร์ Short ของรายย่อยทั้งหมดกลายเป็นสภาพคล่องให้สถาบันไล่ราคาขึ้นไป (Short Squeeze) จนราคาทะลุกลับไปยังจุดเริ่มต้นที่เริ่มดึง Fibo อย่างรวดเร็ว
