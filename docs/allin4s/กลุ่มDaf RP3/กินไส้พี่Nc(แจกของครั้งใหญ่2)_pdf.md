## 2. ส่วนของกราฟแท่งเทียน (Candlestick Chart Analysis)
วิเคราะห์รูปแบบและพฤติกรรมราคาจากภาพกราฟ:

**1. รูปแบบ Twin Wick Reversal (TWR) และ Wick Assassin (WA):**
- **TWR:** กราฟทำไส้เทียนยาวๆ ในระดับเดียวกันหรือใกล้เคียงกัน 2 ครั้ง (ไส้คู่) บ่งบอกถึงการปฏิเสธราคา (Rejection) ที่รุนแรง ผู้สอนแนะนำให้ใช้จังหวะนี้ในการ "สวนเทรนด์ (Reversal)"
- **WA:** กราฟทำท่าย้อนกลับมาเหมือนจะกินไส้เดิม (Sweep Liquidity) แล้วทิ้งไส้หลอกไว้ ก่อนที่จะกลับตัวไปในทิศทางตรงกันข้ามอย่างรุนแรง

**2. การวิเคราะห์แท่งกลืนกิน (Engulfing) และ ตำหนิ:**
- **แท่งกลืนกิน (Engulfing):** เป็นจุดยืนยันการกลับตัวที่แข็งแกร่ง แต่ผู้สอนเน้นว่าต้องดูด้วยว่าแท่งกลืนกินนั้นสามารถ "เบรกไส้ (Break Wick)" หรือ "เบรกเนื้อ (Break Body)" ของสวิงก่อนหน้าได้หรือไม่ ถ้าเบรกไม่ได้ อาจเป็นเพียงการเด้งเพื่อลงต่อ
- **ตำหนิ:** ร่องรอยที่กราฟทิ้งไว้ เช่น แท่งเทียนที่ปิดไม่สมบูรณ์ หรือทิ้งไส้แปลกๆ กราฟมักจะย้อนกลับมา "เคลียร์ตำหนิ" เสมอ

**3. การทำงานร่วมกับ FVG (Fair Value Gap):**
- กราฟมักจะวิ่งเข้าหา FVG เพื่อเติมเต็มสภาพคล่อง (Fill Liquidity) ก่อนที่จะไปต่อตามเทรนด์
- ผู้สอนมักจะวาง Pending Order (Buy Limit / Sell Limit) ไว้ที่บริเวณ FVG หรือจุดที่มี "แท่งตำหนิ" ซ้อนทับอยู่ เพื่อดักการย่อตัว

**4. Multi-Timeframe Analysis:**
- ผู้สอนมักจะให้ดูสวิงหลักใน Timeframe ใหญ่ (H1, H4, D1) และหาจุดเข้าใน Timeframe เล็ก (M5, M15)
- มีการยกตัวอย่างว่า สิ่งที่เป็นแท่งกลืนกินใน M15 อาจเป็นเพียงการทิ้งไส้ใน H1 ดังนั้นการอ่านความสัมพันธ์ระหว่าง Timeframe จึงสำคัญมากในการหาจุดเข้าที่ได้เปรียบ
