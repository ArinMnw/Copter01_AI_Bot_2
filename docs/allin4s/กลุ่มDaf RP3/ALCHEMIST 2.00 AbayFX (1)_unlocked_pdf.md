# ALCHEMIST 2.00 AbayFX (1)_unlocked

เอกสารเล่มนี้ (ความยาวประมาณ 119 หน้า ซึ่งถูกแบ่งเป็น 10 ไฟล์ย่อย) เป็นคู่มือเทรดระบบ **ALCHEMIST + SNR** ของ AbayFX ซึ่งเน้นการผสมผสานระหว่างแนวรับแนวต้าน (SNR) และโครงสร้างตลาด

## สรุปเนื้อหาสำคัญ
- **แก่นของระบบ (Core Components)**:
  1. **SNR Zone**: ไม่ใช่แค่เส้นแนวรับแนวต้านธรรมดา แต่ต้องเป็นโซนสำคัญ
  2. **Liquidity Sweep (Miss)**: ราคาต้องมีการกวาดสภาพคล่อง (Sweep) ทะลุ High หรือ Low ก่อนหน้าเพื่อดึงดูด Stop Loss ของรายย่อย
  3. **Order Block (OB)**: เน้นหาจุดเข้าที่บริเวณกึ่งกลาง (50% Midpoint) ของ OB ล่าสุด
  4. **Market Structure (MS)**: โครงสร้างตลาดหลักและโครงสร้างย่อย
- **เงื่อนไขการเข้าเทรด (Confluence Checklist)**:
  - ทิศทางหลัก (HTF) สนับสนุนการเทรด (เช่น H4 เป็นเทรนด์ขาขึ้น)
  - เกิดการ Sweep สภาพคล่องที่โซนนั้นๆ
  - โซนที่เข้าเทรดต้องมีการซ้อนทับกันของ SNR + OB + 50% OB
  - มีแท่งเทียนคอนเฟิร์ม (Engulfing / Rejection / ปิดกลับเข้ามากรอบเดิม)
- **ข้อผิดพลาดที่ควรระวัง (Common Mistakes)**:
  - เข้าเทรดเพียงเพราะกราฟชนเส้น โดยไม่มีการคอนเฟิร์ม
  - ละเลยการดู Liquidity และ Order Block
  - เทรดสวนเทรนด์หลัก (HTF)
- **เนื้อหาที่เหลือ**: เป็นภาพกราฟตัวอย่างที่แสดงให้เห็นถึงการเกิด Liquidity Sweep, การเกิด Market Structure Shift (MSS) และการย่อกลับมาทดสอบ OB ที่ 50% แล้วไปต่อ
