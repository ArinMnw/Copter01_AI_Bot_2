# การวิเคราะห์และถอดความจากเอกสาร การอ่านเเรงเเท่งเทียน.pdf อย่างละเอียด

เอกสาร `การอ่านเเรงเเท่งเทียน.pdf` (ความยาว 20 หน้า) เน้นสอนวิธีการอ่าน "โมเมนตัม" (Momentum) หรือแรงของแท่งเทียนแบบแท่งต่อแท่ง เพื่อดูว่ากราฟมีแรงไปต่อ หรือหมดแรงและเตรียมจะกลับตัว รวมถึงเทคนิคการเข้าออเดอร์ที่จุด 50% ของเนื้อเทียน

## ส่วนที่ 1: การถอดข้อความ คำอธิบาย (Word-for-word)

**หน้า 1: กฎการอ่านแรงแท่งเทียน**
1.) การอ่านการกลับตัวหรืออ่านการส่งแรง ถ้าเกิดแท่งเทียนเขียวหรือแดง อีกแท่งต้องสีตามนั้นด้วยเสมอ เช่นถ้าจะให้ขึ้นต้องเป็นเขียวอีกแท่ง ถ้าอยากให้ลงก็ต้องแดงอีกหนึ่งแท่งนั่นเอง
2.) ต้องปิดคลุมไส้ของแท่งก่อนหน้าให้ได้ ถ้าปิดไม่ได้ก็จะเป็นการย้อนโครงสร้างขนาดย่อม (ในกรณีที่เป็นตำหนิ) คีย์คือต้องปิดเหนือไส้นั้นเอง (การอ่านโครงสร้างนี้ใช้ไม่ได้กับการเกิดสภาพเคลียร์)

**หน้า 2-5: กฎ 50% ของเนื้อเทียน**
- คีย์คือต้องปิดเหนือไส้และจบเขียว (ฝั่งB)
- 50%ของเนื้อแท่งเขียว / 50%ของเนื้อแท่งแดง (เป็นการย้อนด้วยแท่งเทียน) ฝั่งS
- คลุมไส้แท่งก่อนหน้าได้
- ถ้าลงต้องจบแดงอีกแท่ง และปิดคลุมไส้แท่งก่อนหน้าได้
- ถ้าขึ้นต้องจบเขียวอีกแท่ง แต่ตรงนี้จบแดง เหมือนกันกับภาพปัจจุบันนี้มั้ย?

**หน้า 6-9: การอ่านแรงที่หมดลง (ปิดไม่คลุมไส้)**
- แท่งนี้คลุมไส้แท่งก่อนหน้าได้
- ทำไมผมถึงดักตรงนี้? แล้วไม่ดักตรงกลางของแท่งเขียวล่ะ?
- ตรงนี้เป็นDMของH1ซึ่งผมไม่เล่น ถ้าขึ้น แท่งนี้ ต้องจบเขียว เห็นจบแดงผม เลยSลงไปซะเลย
- ผมเข้าด้วยโดจิแดง แท่งนี้ปิดไม่คลุมไส้ ผมรู้แล้วว่าไม่ขึ้น

**หน้า 13-16: ตำหนิซ้อนทับ และการโค้งลง**
- ตำหนิซ้อนทับ เข้าที่50%ของเนื้อแท่งเขียว (M15 / M30)
- ขึ้นได้เพราะ แท่งที่2จบเขียว และปิดคลุมไส้แท่งที่1
- ปิดไม่คลุม ไส้แท่งก่อนหน้า มันจะค่อยๆโค้งลง
- ทำไมแท่งนี้ถึงย้อนกลับได้? ย้อนกลับมาที่ครึ่งหนึ่ง ของแท่งเขียวตัวเอง50%
- ถ้าขึ้นต้องจบเขียวอีกแท่ง และคลุมไส้แท่งเขียวก่อนหน้าได้ เห็นหรือไม่ว่ามันไม่ขึ้น

**หน้า 17-20: ด่านกลืนกินและการกลับตัว**
- ติดตรงนี้ของM5นี่เอง ที่เรียกกันก็คือเขียวกลืนกิน
- การอ่านการย้อนของกราฟจะเป็นแบบนี้ เมื่อเกิดการกลับตัวของฝั่งตรงข้ามและย้อนกลับมา มันจะติดตรงเขียวกลืนกินก่อนถ้าผ่านตรงเขียวกลืนกิน มันจะวิ่งไปหาตำหนิทันทีนั่นเอง
- เกิดการกลับตัวฝั่งSที่แข็งแรงกว่า และปิดคลุมไส้แท่งเขียวได้และเป็น2H จริงๆไม่จำเป็นต้อง2H เห็นการกลับตัวฝั่งไหนฝั่งหนึ่ง ให้เล่นตามน้ำไปเลย
- ย้อนกลับมาแล้ว ถ้าดักไปทำไมแหกละ
- เกิดขึ้น เฉพาะขาลง ย้อน50%ของแท่งแดงก่อนหน้า ปิดไม่คลุมไส้แท่งก่อนหน้า

## ส่วนที่ 2: การวิเคราะห์พฤติกรรมราคา (Price Action) และกราฟแท่งเทียน

เอกสารนี้เจาะลึกวิชา "แท่งเทียน" โดยมีกฎเกณฑ์ที่ตายตัวและสามารถนำไปประยุกต์ใช้ในการหาจุดเข้า (Entry) ได้อย่างเฉียบคม:

### 1. กฎการส่งแรง (Momentum Confirmation)
การที่กราฟจะเป็นเทรนด์และวิ่งไปต่อได้อย่างแข็งแรง ต้องประกอบด้วยเงื่อนไข 2 ข้อ:
- **ต้องมาเป็นคู่สีเดียวกัน:** ถ้าจะขึ้น แท่งแรกเขียว แท่งที่สองก็ต้องปิดเขียว (ถ้าแท่งสองปิดแดง = เสียทรง)
- **ต้อง "ปิดคลุมไส้" ให้ได้:** แท่งที่สอง ไม่ใช่แค่ปิดเป็นสีเดียวกัน แต่เนื้อเทียนของแท่งที่สอง **จะต้องปิดสูงกว่า (หรือต่ำกว่าในขาลง) "ปลายไส้" ของแท่งแรกเสมอ** 
- หากแท่งที่สองปิดสีเดียวกัน แต่เนื้อเทียน **"ปิดไม่คลุมไส้แท่งแรก"** แสดงว่ากราฟ **หมดแรง (Exhaustion)** และจะค่อยๆ โค้งตัวกลับทิศทาง (ย้อนโครงสร้าง) แท่งแบบนี้เรียกว่า **"แท่งตำหนิ"**

### 2. กฎการเข้าออเดอร์ที่ 50% ของเนื้อเทียน (The 50% Body Retracement)
เมื่อกราฟเกิดการส่งแรงที่ถูกต้อง (แท่งยาว หรือ กลืนกิน) มักจะมีการย่อตัว (Pullback) กลับมาเพื่อทดสอบแรงก่อนไปต่อ
- **จุดเข้าที่แม่นยำ:** ให้วัดระยะเฉพาะ "เนื้อเทียน" (Body) ของแท่งที่ส่งแรง (ไม่รวมไส้)
- กราฟมักจะย่อตัวลงมาแตะที่ระดับ **50% ของเนื้อเทียน** พอดีเป๊ะ แล้วดีดกลับไปตามเทรนด์เดิม
- เราสามารถใช้ระดับ 50% ของเนื้อเทียนสีเขียว (เพื่อ Buy) หรือ 50% ของเนื้อเทียนสีแดง (เพื่อ Sell) เป็นจุดวาง Pending Order ได้เลย

### 3. กรณี "ตำหนิซ้อนทับ" (Overlapping Defects)
- เมื่อกราฟย่อตัวลงมาและเกิดการทำ "ปิดไม่คลุมไส้" (แท่งตำหนิ) ซ้ำๆ กัน 2-3 ครั้งในบริเวณเดียวกัน 
- นั่นแปลว่ามีแนวรับที่แข็งแกร่งมากรองรับอยู่ (Liquidity build-up)
- จุดเข้าที่ดีที่สุดในกรณีนี้ ก็ยังคงเป็นการดักเข้าที่ระดับ **50% ของเนื้อแท่งเขียว** ที่เป็นตัวรับแรงบริเวณนั้น

### 4. ด่านสกัดกั้นตอนกราฟย้อนกลับ
- เมื่อกราฟเกิดการกลับตัว (เช่น เกิด 2H แล้วทุบลงมา)
- ด่านแรกที่กราฟจะต้องเจอและมักจะเด้ง (ติดด่าน) คือ **"แท่งกลืนกิน"** (Engulfing Candle) ในอดีต
- แต่ถ้าราคาเททิ้งแรงจนสามารถทะลุ "แท่งกลืนกิน" ลงมาได้ เป้าหมายต่อไปที่กราฟจะพุ่งไปหาอย่างรวดเร็วคือ **"แท่งตำหนิ"** (แท่งที่ปิดไม่คลุมไส้) ที่ถูกทิ้งร่องรอยไว้ด้านล่างนั่นเอง
