# การวิเคราะห์และถอดความจากเอกสาร ขั้นตอนในการอ่านกราฟก่อนจะเข้าเทรด.pdf อย่างละเอียด

เอกสาร `ขั้นตอนในการอ่านกราฟก่อนจะเข้าเทรด.pdf` (ความยาว 18 หน้า) เป็นการนำความรู้ทั้งหมด (Fibo, แท่งเทียน, 2H/2L, จิตวิทยาตัวเลข) มารวมกันเป็นขั้นตอน (Step-by-step) เพื่อใช้วิเคราะห์กราฟจริงก่อนตัดสินใจเข้าออเดอร์

## ส่วนที่ 1: การถอดข้อความ คำอธิบาย (Word-for-word)

**หน้า 1-4: การเตรียม Fibo และหาเป้าหมาย**
- ภาพวาด รายการโปรด ขั้นแรกหยิบ Fibo ขึ้นมา
- ขั้นที่2 ตีแท่งกลืนกิน ที่สวิงหลัก แท่งใหญ่ๆแบบนี้ ผมจะไม่มองไกลถึง RUN ผมจะมองแค่ KRH2
- และการคอนเฟิร์มว่าจุดนี้คือการคอนเฟิร์มแบบ100% คือสวิงรอง ก็อยู่ โซน RUN
- ซึ่งตีมาแล้ว เป้า2 จะตก 3709 แต่อย่างที่รู้กัน กราฟต้องการจะพักตัว มันจะไม่แตะต้องเลข 5และ0 และเลขจิตวิทยาที่ผมใช้ จะเป็นเลข 7
- 3707 คือเป้าหมายของเรา
- การคอนเฟิร์มเข้า3707ว่าลงแน่ คือการกินไส้ของD1

**หน้า 5-9: การดูไส้เทียนและแนวรับ**
- ไส้นี้>>> ซึ่งวันที่ข่าวจะออก กราฟยังไม่ได้มีการกินไส้เขียวของแท่งก่อนหน้าให้เห็น (ก.ย.)
- การที่กราฟจะลงได้แรงๆ จะเห็นได้ชัดเลยว่าไส้แดงจะยาวกว่าไส้เขียว 
- ฝั่งB ไส้เขียวก็จะยาวกว่าไส้แดง
- แล้วออเดอร์จุดที่ผมเห็น 3630 มายังไง? จุดนี้มีการสะสมอย่างมหาศาล <<และLowนี้ กำลังซับพอร์ตทั้งหมดอยู่ 
- หลุดจุดนี้เป็นขาลงระยะสั้นทันที ซึ่งจริงๆแล้วต้องเข้าที่27-28 อย่างที่รู้กันว่าผมไม่เข้า ตระกูลเลข0อยู่แล้ว เลข7-8 จะดีกว่า

**หน้า 10-16: การคอนเฟิร์มด้วย TF เล็ก และ 2H/2L**
- ตอนที่เราเห็นแค่นี้ ของTF H4 เราจะมั่นใจได้ยังไงว่าแท่งต่อไปจะไม่ไปกินไส้ ของแท่งที่แล้ว
- ง่ายๆเลย TF H1 หรือ TF ที่เล็กลง ชิงสร้างจุดกลับตัวก่อน หรือก็คือ 2L
- อย่างที่รู้กัน 2H ไม่หลุด Low ให้ Buy กลับขึ้นไป (ปิดไม่คลุมไส้ ย้อน Low)
- การคอนเฟิร์มถือต่อของผม (การกลับตัวจริง) SP FVG L1 L2
- จุดเริ่มต้นของการกลับตัว มาจากFVG รับและไม่หลุด (M30)
- โครงสร้างนี้เหมือน H4 หรือไม่? ไม่หลุดLow Buy ขึ้นไป
- 2H ไม่หลุด Low เกิดเป็น 2L แทน เพราะฉะนั้น โฟกัสตามความจริงอย่าจินตนาการ กราฟเส้นจะเห็นได้ชัดเลยว่ามันไม่ได้หลุดLow

**หน้า 17-18: การคอนเฟิร์มขั้นสุดท้าย**
- นิสัยของแท่งเทียนจะวิ่งหาไส้ที่ยาวเสมอ
- และการคอนเฟิร์มสุดท้าย ที่ผมถือแบบใจเย็น
- 2L ไส้ยาว H12 จบเขียว ไม่มีการคอนเฟิร์มที่ดีกว่านี้แล้ว

## ส่วนที่ 2: การวิเคราะห์พฤติกรรมราคา (Price Action) และขั้นตอนการเทรด

เอกสารนี้เรียบเรียงลำดับความคิด (Mindset & Workflow) ในการวิเคราะห์กราฟก่อนเทรดไว้ดังนี้:

### Step 1: กาง Fibo หาเป้าหมายที่แท้จริง
- หาสวิงหลักแล้วกาง Fibo ที่แท่งกลืนกิน/แท่งตำหนิ
- **กฎเหล็ก:** ถ้าแท่งเทียนที่ใช้เป็นฐานกาง Fibo มีขนาด "ใหญ่เกินไป" ห้ามตั้งเป้าหมายไปที่ระดับ RUN (7.044) เพราะมันไกลเกินไป ให้มองเป้าหมายแค่วาง TP ไว้ที่ระดับ **KRH2 (3.097)** ก็พอ
- **Double Confirmation:** หา "สวิงรอง" (สวิงเล็กที่อยู่ในสวิงหลัก) แล้วกาง Fibo ซ้อนทับไปอีกชั้น หากเป้า RUN ของสวิงรอง ไปตกที่เดียวกับเป้า KRH2 ของสวิงหลักพอดี นั่นคือโซนเป้าหมายที่แม่นยำ 100%

### Step 2: ปรับเป้าหมายด้วย "เลขจิตวิทยา" (Psychological Numbers)
- สมมติว่าเป้า Fibo วัดได้ที่ราคา 3709
- ระบบนี้ **"ห้ามตั้ง TP/Limit ที่ตัวเลขลงท้ายด้วย 0 หรือ 5"** (เช่น 3705, 3710) เพราะเป็นจุดที่รายย่อยชอบตั้งกัน และเจ้ามือมักจะวิ่งไปไม่ถึง (ตกรถ) หรือวิ่งทะลุไปเลย
- ให้ใช้ตัวเลขจิตวิทยาที่ลงท้ายด้วย **7 หรือ 8** แทน (เช่น 3707 หรือ 3708) เพื่อชิงความได้เปรียบ

### Step 3: หาการ "กินไส้" ใน Timeframe ใหญ่ (D1)
- ดู TF ใหญ่สุดอย่าง D1 ว่ามีไส้เทียนของวันก่อนหน้าที่ยัง "ปิดไม่คลุม" (ยังไม่ถูกกิน) อยู่หรือไม่
- ถ้ามี กราฟจะมีแนวโน้มวิ่งไปกินไส้นั้นอย่างแน่นอน
- ข้อสังเกตแรงวิ่ง: ในเทรนด์ขาลงที่แข็งแกร่ง เราจะสังเกตเห็นเลยว่า "ไส้เทียนสีแดงจะยาวกว่าไส้สีเขียวเสมอ" (และสลับกันในขาขึ้น)

### Step 4: ดูโครงสร้างขัดแย้งใน TF เล็ก
- หาก TF ใหญ่ (เช่น H4) บอกว่าน่าจะต้องย่อตัวลงมาเก็บไส้เทียน แต่เราจะรู้ได้อย่างไรว่ามันจะไม่ย่อ?
- ให้ซูมเข้าไปดู TF H1 หรือ M15 หากใน TF เล็กกราฟชิงทำโครงสร้าง **2L (Double Low)** ไปเสียก่อน นั่นแปลว่า **"กราฟยกเลิกการย่อตัว และพร้อมจะพุ่งขึ้นทันที"**

### Step 5: การแก้เกมส์ของ 2H / 2L
- สมมติว่ากราฟทำโครงสร้าง **2H (Double High)** ซึ่งโดยปกติแปลว่ากราฟต้อง "ทุบลง"
- แต่ถ้ามันทุบลงมาแล้ว **"ไม่หลุด Low ล่าสุด"** (ไม่สามารถทำ New Low ได้) 
- โครงสร้าง 2H นั้นจะพังทลาย และมันจะแปลงร่างกลายเป็นโครงสร้าง **2L (Double Low)** แทนทันที
- **คำแนะนำ:** อย่าจินตนาการไปเองว่ามันต้องลงตาม 2H ให้ดูความจริง ถ้ามันไม่หลุด Low ให้เปลี่ยนหน้าเล่นเป็น Buy สวนขึ้นไปทันที (ใช้กราฟเส้น Line Chart เพื่อดู Low ได้ชัดเจนขึ้น)

### Step 6: การคอนเฟิร์มไม้ตายสุดยอด (Ultimate Confirmation)
- นิสัยของแท่งเทียนคือ **"ชอบวิ่งไปหาไส้เทียนที่ยาวที่สุดเสมอ"**
- สัญญาณที่แข็งแกร่งและแม่นยำที่สุดที่ทำให้ถือออเดอร์ได้อย่างสบายใจ คือการเกิดโครงสร้าง **2L ที่มีไส้ยาวๆ และจบด้วยแท่งสีเขียว ใน Timeframe ใหญ่ระดับ H12 หรือ D1** (ไม่มีการคอนเฟิร์มไหนดีกว่านี้แล้ว)
