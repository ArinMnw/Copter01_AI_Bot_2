## ส่วนที่ 2: การวิเคราะห์พฤติกรรมราคา (Price Action) และกราฟแท่งเทียน

เอกสารฉบับนี้อธิบายเพิ่มเติมจากเรื่อง Fibonacci พื้นฐาน โดยลงลึกถึง "ความถูกต้อง" ในการกาง Fibo และ "การยืนยัน" พฤติกรรมราคาเมื่อถึงเป้าหมาย:

### 1. กฎการกาง Fibonacci (Anchor Selection)
- **สวิงหลัก (Main Swing):** ต้องกาง Fibo จากแท่งเทียนที่เกิดจากการดันราคากลับตัว หรือจากสวิงหลักใน TF ที่ใหญ่กว่า (เช่น H1) ถ้าระยะสั้นอาจใช้สวิงของ M15 มาเทรดใน M5
- **การเลือกแท่งเทียนอ้างอิง:** 
  - **ห้าม** ใช้แท่งกลืนกิน (Engulfing) ที่มีเนื้อเทียน "ยาวใหญ่เกินไป" เป็นฐานในการกาง 0-1 เด็ดขาด เพราะจะทำให้เป้าหมายปลายทาง (TP) เพี้ยนไปไกล
  - **ควร** ใช้แท่งกลืนกินที่มีขนาดพอประมาณ 
  - **ดีที่สุด** คือการใช้ "แท่งตำหนิ" (ปิดไม่คลุมไส้) ที่มี "ไส้ยาวๆ" เป็นฐาน Fibo จะให้ความแม่นยำสูงที่สุด

### 2. กฎการนับว่า "เข้าเทสโซน" (Valid Test)
- กราฟจะถูกนับว่าไปชนระดับ Fibo (เช่น ระดับ 3 หรือ 5.165) อย่างแท้จริงก็ต่อเมื่อ **"เนื้อเทียน" (Body)** ปิดเข้าไปในโซนหรือปิดบนเส้น Fibo นั้น
- หากกราฟพุ่งทะลุเส้นไปแค่ไส้ แล้วเนื้อเทียนรูดกลับมาปิดต่ำกว่าด่านด้านล่าง (เช่น ทะลุ 5.165 แต่รูดปิดใต้ 3.097) ระบบจะ **ไม่นับ** ว่ากราฟไปเทสโซน 3 แล้ว

### 3. รูปแบบการทำกำไร (Fibo Models)
เอกสารยกตัวอย่าง Model พฤติกรรมราคาที่พบบ่อย (คล้ายกับในไฟล์ Fibo.pdf เดิมแต่ชัดเจนขึ้น):
- **Model 2 Test 50%:** กราฟขึ้นไปชนระดับ 2 (3.097) แล้วร่วงลงมาทดสอบโซน 50%-60% (0.382 - 0.57) หากรับอยู่จะวิ่งกลับขึ้นไปเป้า RUN
- **Model 3 Test 1:** กราฟขึ้นชนระดับ 3 (5.165) แล้วร่วงลงมาลึกถึงระดับ 1 (1.617) หากรับอยู่จะพุ่งกลับไปหา RUN

### 4. ด่านชี้ชะตา: ระดับ 2 (KRH2 / 3.097)
ระดับ 3.097 เป็นเส้น Support & Resistance ที่สำคัญที่สุดในการดูว่ากราฟจะจบรอบหรือไปต่อ:
- **"ยืน 2 ได้":** หากกราฟร่วงลงมาจากเป้า RUN แล้วลงมาแตะ 3.097 (ระดับ 2) และสามารถพักตัวสร้างฐานปิดเหนือเส้นนี้ได้ กราฟจะดีดกลับไปหาเป้า RUN อีกครั้งเป็นเส้นตรง
- **"ยืน 2 ไม่ได้":** หากกราฟร่วงหลุดเส้น 3.097 ลงไป และเมื่อเด้งกลับมาเทสเส้นนี้แต่ไม่สามารถผ่านขึ้นไปได้ (ไม่ทำ High ใหม่) กราฟจะถูกตบลงอย่างรุนแรง และไหลทะลุจุด 0 (จุดเริ่มต้นสวิง) ลงไปทำ New Low เสมอ

### 5. เป้าหมาย RUN
- ค่า Fibo ของเป้า RUN จะต่างกันเล็กน้อยขึ้นอยู่กับแท่งอ้างอิงที่เราตี:
  - ถ้าตีจาก **แท่งกลืนกิน** เป้า RUN = 7.044
  - ถ้าตีจาก **แท่งตำหนิ** เป้า RUN = 7.467
