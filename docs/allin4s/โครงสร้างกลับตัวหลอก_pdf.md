# โครงสร้างกลับตัวหลอก (Fake Reversal Structure)

ไฟล์สไลด์อธิบายเทคนิคการสังเกต "โครงสร้างหลอกกลับตัว" ของแท่งเทียน เพื่อดูว่ากราฟกำลังจะไปทิศทางไหนต่อ

## สรุปเนื้อหาสำคัญ
- **ตำแหน่งที่เกิด**: โครงสร้างหลอกมักเกิดขึ้นระหว่างทางตรงกลางของเทรนด์ (เช่น มี H-L แล้วกำลังขึ้น) คำถามสำคัญคือ กราฟกำลังขึ้นเพื่อลง หรือขึ้นต่อ?
- **จุดสังเกตลักษณะเด่น**:
  - จะมีแท่งเทียนสีแดงทิ้งตัวลงมา แต่ไม่สามารถปิดคลุมไส้แท่งก่อนหน้าได้เลยประมาณ 2-3 แท่ง
  - แต่จะมีอยู่แท่งหนึ่งที่สามารถลงมาปิดคลุมไส้ (กินไส้) ได้สำเร็จ 
  - **จุดหลอก (Fake-out)**: ปัญหาคือหลังจากที่มันปิดคลุมไส้ได้ มันกลับถูกงัดขึ้นมาปิดเป็นแท่งเขียว (หรือทิ้งไส้ยาวด้านล่าง) ซึ่งแท่งที่ทำหน้าที่ปิดคลุมไส้ได้นั้น คือการสร้างโครงสร้างหลอกว่าจะลง เพื่อเตรียมบินขึ้นต่อ หรือเป็นการทำโครงสร้างเพื่อที่จะทุบลงในภายหลัง
- **ภาพประกอบ**: มีการแสดงลำดับแท่งเทียนที่ค่อยๆ ลงมาทีละแท่ง (ไม่คลุมไส้) จนกระทั่งมีแท่งที่คลุมไส้ได้ แต่ถูกงัดกลับขึ้นเป็นสีเขียวทันที ซึ่งเป็นสัญญาณว่าการลงนั้นไม่แข็งแรงจริง (เป็นโครงสร้างหลอก)
