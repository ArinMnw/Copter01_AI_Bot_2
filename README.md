# Copter01_AI_Bot

บอทเทรดอัตโนมัติสำหรับ MT5 ที่ควบคุมผ่าน Telegram

## ภาพรวม

- สแกน pattern หลาย timeframe
- เปิด pending order และ market order ตามเงื่อนไขของแต่ละท่า
- จัดการ SL / TP / trail / sweep / pending cancel หลังเปิด position

## วิธีรัน

```bash
python main.py
```

บน Windows:

```bash
run.bat
```
