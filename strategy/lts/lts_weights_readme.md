# คำอธิบายไฟล์ Weights ทั้งหมดของระบบ LTS

ไฟล์เหล่านี้คือ "สมอง" ที่คอยกำหนดสัดส่วนการออก Lot (น้ำหนักความสำคัญ) ให้กับแต่ละกลยุทธ์ย่อย (Leg) ภายในพอร์ต LTS ของเราค่ะ โดยแต่ละไฟล์ถูก Optimize (คำนวณหาค่าที่ดีที่สุดด้วย AI) มาเพื่อจุดประสงค์ที่ต่างกัน

---

## 1. 💎 strategy/lts/optimized_weights/lts71_optimized_weights.txt (โหมดปลอดภัย 100% - แนะนำสำหรับการปั้นพอร์ตจริง)
**จุดประสงค์:** สร้างระบบทบต้น (Compounding) ที่เสถียรที่สุด โดยปราศจากความเสี่ยงล้างพอร์ต
- **จำนวนท่าที่ใช้:** 71 ขา (คัดมาเฉพาะท่าที่แม่นยำสูงและเสริมกันเพื่อลบ Drawdown)
- **สถิติเฉลี่ย:** กำไรประมาณ **1.46% ต่อวัน**
- **ความเสี่ยง:** **0.00% Max Drawdown** (ในสถิติ 550 วันย้อนหลัง หักล้างกันจนไม่มีวันไหนที่พอร์ตปิดลบเลย)
- **เป้าหมาย 10,000$/วัน:** ทุน $1,000 จะทบต้นไปถึงกำไรวันละ $10,000 ได้ในเวลาประมาณ **451 วันเทรด (เกือบ 2 ปี)** 
- **การใช้งาน:** เหมาะที่สุดสำหรับการนำไปรันพอร์ตเงินจริงทิ้งไว้ระยะยาว ปล่อยให้มันค่อยๆ โตอย่างมั่นคง

## 1.5. 🛡️ strategy/lts/optimized_weights/lts250_optimized_weights.txt (โหมดปลอดภัยระดับอัปเกรด - LTS 71 Extended)
**จุดประสงค์:** ต่อยอดจาก LTS 71 โดยเพิ่มจำนวนขาเป็น 250 ขา เพื่อเพิ่มเพดานกำไรให้สูงขึ้นในขณะที่ยังคุมความเสี่ยงให้ต่ำมาก
- **จำนวนท่าที่ใช้:** 250 ขา
- **สถิติเฉลี่ย:** ล็อกเป้าที่กำไร **$2,000 ต่อวัน**
- **ความเสี่ยง:** Worst day อยู่ที่ประมาณ -$956 ถือว่าปลอดภัยมากเมื่อเทียบกับกำไร
- **การใช้งาน:** เหมาะกับคนที่พอใจกับความปลอดภัยของ LTS 71 แต่อยากได้ความเร็วและจำนวน Order ที่มากขึ้น



## 3. 🛡️ strategy/lts/optimized_weights/lts44_optimized_weights.txt (พอร์ตมาตรฐานเดิม)
**จุดประสงค์:** พอร์ตขนาดกลางสำหรับทุนระดับพันเหรียญ
- **จำนวนท่าที่ใช้:** 44 ขา
- **สถิติ:** ทำกำไรเฉลี่ยได้ดี แต่อาจจะมี Drawdown ในวันที่เลวร้ายที่สุด (Worst day) อยู่ที่ราวๆ -$1,400 
- **การใช้งาน:** เหมาะสำหรับคนที่เผื่อทุนรันไว้ประมาณ $2,000 - $3,000 จะทนแรงสวิงของตลาดได้สบายๆ

## 3.5. 🚀 strategy/lts/optimized_weights/lts150_optimized_weights.txt (พอร์ตมาตรฐานระดับอัปเกรด - LTS 44 Extended)
**จุดประสงค์:** ต่อยอดจาก LTS 44 โดยเพิ่มจำนวนขาเป็น 150 ขา ช่วยให้เส้นกราฟเนียนขึ้นและกำไรสูงขึ้น
- **จำนวนท่าที่ใช้:** 150 ขา

- **การใช้งาน:** สามารถใช้งานแทน LTS 44 เดิมได้เลย เพราะสัดส่วน Drawdown/Profit ดีกว่ามาก
