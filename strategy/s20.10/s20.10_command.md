# S20.10 Backtest Commands

รันคำสั่งนี้ใน Terminal เพื่อทดสอบ Backtest สำหรับ S20.10 (Multi-timeframe 30, 60, 90, 120, 180 วัน):

```bash
python strategy/s20.10/backtest-sim/backtest_S20_10_runner.py --days 0 --tf all
```

*(หมายเหตุ: `--days 0` หมายถึงให้รันลูป 30, 60, 90, 120, 180 วันอัตโนมัติ และ `--tf all` คือสแกนทุก Timeframe)*
