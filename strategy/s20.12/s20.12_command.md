# S20.12 Command Reference

หน้านี้รวบรวมคำสั่งสำหรับใช้งานสคริปต์ Backtest ของกลยุทธ์ `S20.12 FutureKey` (Candlestick Mechanics) ครับ คุณสามารถก๊อปปี้คำสั่งในกรอบสีดำไปรันที่ Terminal ได้เลย

---

### 1. `backtest_S20_12_runner_mt5.py`
สคริปต์ Backtest พื้นฐานที่ดึงข้อมูลจาก MT5 โดยตรง สามารถระบุจำนวนวัน (`--days`), Timeframe (`--tf`), และการจัดการความเสี่ยง (`--compound`) ได้อย่างอิสระ

**คำสั่งรันแบบมาตรฐาน (แนะนำให้บันทึก log):**
```bash
python -u strategy/s20.12/backtest-sim/backtest_S20_12_runner_mt5.py --days 30 --tf all --compound 2 > bt_s20_12_out.log 2>&1
```
*(คำสั่งนี้จะทำการเทส 30 วัน, สแกนทุก Timeframe, ทบต้นที่ 2% และบันทึกผลลัพธ์ไว้ในไฟล์ `bt_s20_12_out.log`)*

**คำสั่งรันแบบดูผลลัพธ์บนจอโดยตรง:**
```bash
python strategy/s20.12/backtest-sim/backtest_S20_12_runner_mt5.py --days 30 --tf all --compound 2
```

*ตัวอย่างพารามิเตอร์เพิ่มเติม:*
- `--days 60` (ระบุจำนวนวันที่ต้องการเทสย้อนหลัง)
- `--tf M15` (ระบุ Timeframe เจาะจง เช่น M1, M5, H1, D1 หรือ all)
- `--compound 1.0` (ระบุเปอร์เซ็นต์ความเสี่ยง/พอร์ต สำหรับคำนวณการทบต้น Lot Size)
