# S20.8 Command Reference

หน้านี้รวบรวมคำสั่งสำหรับใช้งานสคริปต์ Backtest ทั้ง 6 ไฟล์ในโฟลเดอร์ `strategy/s20.8/backtest-sim/` ครับ คุณสามารถก๊อปปี้คำสั่งในกรอบสีดำไปรันที่ Terminal ได้เลย

---

### 1. `backtest_s20_8_multi.py` (แนะนำ ⭐)
ใช้สำหรับรันทดสอบทีเดียวครอบคลุมช่วงเวลา **30, 60, 90, 120, 180 วัน** อัตโนมัติ โดยจะแสดงเฉพาะตารางผลลัพธ์สรุปรวม (Summary) เพื่อประหยัดเวลาและดูภาพรวมได้ง่ายที่สุด

```bash
python strategy/s20.8/backtest-sim/backtest_s20_8_multi.py
```

---

### 2. `backtest_S20_8_runner_mt5.py`
สคริปต์ Backtest พื้นฐานที่ดึงข้อมูลจาก MT5 โดยตรง สามารถระบุจำนวนวัน (days) และ Timeframe (tf) ได้เจาะจง เหมาะสำหรับเจาะดูผลลัพธ์ทีละ Timeframe

```bash
python strategy/s20.8/backtest-sim/backtest_S20_8_runner_mt5.py --days 30 --tf all
```
*ตัวอย่างพารามิเตอร์เพิ่มเติม:*
- `--days 60` (จำนวนวัน)
- `--tf M15` (ระบุ Timeframe เช่น M1, M5, H1, D1)
- `--start "01-01-2026 00:00" --end "31-01-2026 23:59"` (ระบุช่วงเวลาชัดเจน)

---

### 3. `sim_s20_8_backtest.py`
สคริปต์สำหรับจำลอง Backtest แบบรวดเร็ว (Fast Simulation) โดยสามารถปรับจูนค่าตัวแปรอื่นๆ เพิ่มเติมได้ เช่น ค่า Spread หรือ Fibo

```bash
python strategy/s20.8/backtest-sim/sim_s20_8_backtest.py --days 30 --tf all
```
*ตัวอย่างพารามิเตอร์เพิ่มเติม:*
- `--spread 0.20` (ปรับแต่งค่า Spread)
- `--fibo 0.5` (ปรับแต่งระดับ Fibo)

---

### 4. `backtest_S20_8_advanced.py`
สคริปต์ Backtest ขั้นสูงที่เพิ่มฟีเจอร์เจาะลึกสถานะของระบบ เหมาะสำหรับการทำ Optimization

```bash
python strategy/s20.8/backtest-sim/backtest_S20_8_advanced.py
```

---

### 5. `export_mt5_history.py`
สคริปต์สำหรับดูดข้อมูลประวัติราคา (Historical Data) จาก MT5 ออกมาบันทึกเก็บไว้ (มักใช้เป็นไฟล์ .csv) เพื่อนำไปใช้งานหรือวิเคราะห์ภายนอก

```bash
python strategy/s20.8/backtest-sim/export_mt5_history.py
```

---

### 6. `compare.py`
สคริปต์สำหรับเปรียบเทียบผลลัพธ์ (Compare results) มักใช้เช็คความถูกต้องระหว่างโค้ดเวอร์ชันเก่าและเวอร์ชันใหม่

```bash
python strategy/s20.8/backtest-sim/compare.py
```
